XMSLEEP 是一款專業的白噪音和自然聲音播放應用，致力於為您提供高質量的音頻體驗。應用內建多種精選的自然聲音，包括雨聲、雷聲、篝火聲、鳥鳴聲等，幫助您放鬆心情、提高專注力、改善睡眠質量。

應用採用 Material Design 3 設計規範，介面簡潔美觀，操作流暢直觀。支援多語言切換、主題自訂、音量獨立調節、固定聲音區域、收藏功能等豐富特性，讓您享受個人化的聲音體驗。

功能特性：
• 多種白噪音：提供雨聲、篝火、雷聲、貓咪呼嚕、鳥鳴、夜蟲等多種自然聲音
• 網路音頻：支援從 GitHub 動態載入更多音頻資源
• 無縫循環：音頻支援無縫循環播放，提供沉浸式體驗
• 音量控制：支援單獨調節每個聲音的音量，或一鍵調整所有聲音
• 精美動畫：內建聲音配有 WebP 動畫，增強視覺體驗
• Material Design 3：採用最新的 Material Design 3 設計規範
• 主題切換：支援淺色/深色模式切換，適配系統主題
• 自訂主題：多種顏色主題可選，支援動態顏色
• 倒計時功能：設置自動停止播放的時間，幫助您控制使用時長
• 預設播放區域：支援將常用聲音添加到預設播放區域，快速訪問
• 收藏功能：收藏喜歡的白噪音聲音
• 全域浮動按鈕：顯示正在播放的聲音，支援快速暫停和展開查看
• 自動更新：支援通過 GitHub Releases 自動檢查更新

隱私說明：
本應用非常重視您的隱私保護。應用不會收集任何個人身份資訊。應用僅收集匿名使用統計數據（如啟動次數、使用時長等），用於改進應用體驗。統計數據僅用於應用功能優化，不會用於任何商業目的或與第三方分享。
